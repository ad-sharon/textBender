# Textbender

Textbender is an AI-powered text processor built with React and Chrome's built-in AI APIs. It helps you detect languages, translate text into your desired language, and summarize content in English — all directly in your browser.

## Features

- **Language Detection** — Automatically identifies the language of any input text
- **Translation** — Translates text into your desired language with ease
- **Summarization** — Condenses text into a clear English summary

## Tech Stack

- **Next.js**
- **Chrome Built-in AI APIs**
- **Tailwind CSS**

## Getting Started

```bash
git clone https://github.com/your-username/textbender.git
cd textbender
npm install
npm run dev
```

## Requirements

- Google Chrome (with Built-in AI APIs enabled)

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

View the deployed application at https://textbender.vercel.app/
